# 이태겸

선린인터넷고등학교 재학 중인 프론트엔드 개발자입니다.
정보보호와 AI에 관심이 많습니다.

---

## 기술

**주력** — TypeScript, React

**사용 가능** — C, Kotlin, Python, NestJS, Node.js, Firebase

---

## 수상

제10회 전국 고등학교 동아리 소프트웨어 경진대회 — 은상 (KAIST 총장상)

---

## 프로젝트

**PrivyFace** — 얼굴정보를 서버에 보내지 않는 신원 인증
ZK-SNARK(영지식 증명)로 얼굴 원본 노출 없이 "나는 DB에 있다"를 증명합니다.
MediaPipe · o1js · Poseidon Hash · Merkle Tree · React
https://github.com/kingcat47/privyface

**Oliver** — 화재 진압 자율주행 로봇 실시간 관제 대시보드 ★ KAIST 총장상
지도 기반 UI, 열화상 영상, WebSocket 실시간 통신. 프론트엔드 전담.
React · WebSocket
https://github.com/oliver-fire/oliver

**CPU Simulator** — C로 만든 모의 CPU + React 실시간 시각화
명령어셋 설계부터 fetch → decode → execute 사이클 구현. 레지스터·메모리 흐름 시각화.
C · React
https://github.com/kingcat47/cpu

**CHISA** — 온디바이스 LLM으로 파일을 정리하는 에이전트
로컬 LLM + RAG로 기기 안에서 파일을 분석·분류·리네이밍. 데이터는 서버로 나가지 않음.
Kotlin · Local LLM · RAG
https://github.com/kingcat47/CHISA
